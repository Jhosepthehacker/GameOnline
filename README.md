## ¿Cómo ejecutar los juegos para lograr una experiencia considerable?

> Instalación De Dependencias

```bash
apt update
apt install python3
apt install build-essential
apt install bash
```

> En Python (app.py)

```bash
python3 app.py
```

> En C++ (app.cpp)

```bash
g++ app.cpp -o app
chmod +x app
./app
```

> En Bash (setup.sh)

```bash
chmod +x setup.sh
./setup.sh
```
